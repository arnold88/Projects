What is PMS?

A Programmable Memory Stick, or PMS, are special pink-colored memory stick that are required to run RCODE, RCodePlus or OpenR SDK programs on AIBO model ERS-2xx , ERS-3xx and ERS-7 series. There is a special encryption program that will identify between the PMS and a normal “General Purpose” memory stick.

How to identify?

Look at the following two pictures, courtesy of www.aibohack.com

 

 

 

Basically if the back of your pink-colored memory stick said something like ERA-MS008 (8MB) or ERA-MS016 (16MB), and without any preprinted label, then it is a PMS.  The one in the CVRL is ERA-MS016.

General Purpose sticks with purple or white color will NOT work.

WARNING: ONLY USE SONY BRAND MEMORY STICK READERS!

WARNING: NEVER DO FULL FORMAT ON A PINK MEMORY STICK UNDER ANY CIRCUMSTANCES!

 

If you format a PMS, you are turning it into a “General Purpose” stick that is useless on the AIBO. Currently, as of March 2009, brand-new 16MB PMS cost around $300 on eBay.

The recommended way to erase a pink memory stick:

    * Open the memory stick drive in Windows

    * Select any folders you see at the root (typically "OPEN-R" and sometimes "CUSTOM")

    * Delete the folders and their contents

    * Empty your recycling bin (if necessary)

This will remove the old files, preparing the stick for a new use.

There is usually a hidden file called MEMSTICK.IND at the root of the memory stick. You do not need to touch it (if you accidentally remove it, don't worry).

ONLY DO A QUICK FORMAT IF ABSOULUTELY NECESSARY.

NEVER DO A FULL FORMAT! EVER!
