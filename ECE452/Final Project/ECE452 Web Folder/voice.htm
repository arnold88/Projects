Operation: AIBO Resurrection

Voice Commands

 

Introduction

 

This program, when properly implemented, is suppose to response to some basic voice commands. I program this using the YART tool. It only contains very few basic commands.

 

Installation Instructions

 

  1. Delete everything on your PMS memory stick. DO NOT FORMAT!
  2. Download the following file and unzip it directly to the ROOT directory (i.e. if your memory stick is assigned drive letter G, then the root directory is G :\) of your memory stick. This is the skeleton of the Open-R SDK.

 

                     R_Code.zip

 

  1. Download the following file and unzip it to the ROOT directory of the memory stick. Overwrite ALL files if prompted.

 

                    voice.zip

 

  1. Now put back the memory stick to the AIBO and turn it on. The R-code boot sequence will begin. Once the AIBO stay idle, you are ready to issue commands.

 

 

Usage Instructions

    

 

 

·  "Take A Picture" - will take a picture

·  "Stand up", "Sit down", "Lie Down"

·  "Go Forward", "Go Backward", "Turn Left", "Turn Right" - start walking

·        "Stop"

·        When walking:

·  "Go Forward", "Go Backward", "Turn Left", "Turn Right", "Stop"

·  "Go Forward" - and he will walk normally

·  "Faster" - and he will walk fast

      ·  "Slow Down" - and he will walk slow

 

Also, the following will occur at any time:      

·  Will meow if it remains idle for too long (over 20 seconds)

·  Will purr if you pet/praise it

·  Will yowl if you hit/scold it

·  When walking, it will stop if it runs into something
