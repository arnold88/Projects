Dancing

 

Introduction

 

Yes you read it right. The AIBO can dance on music. Just keep reading and you will see how it is done.

 

You can see the demo here.

 

The tune is called “Samba De Janeiro”. It is used as the song for goal celebration during the UEFA EURO 2008 soccer tournament. 

 

Right now the program below will respond to only one tune. But obviously you can add additional tune and actions.

 

Installation Instructions

 

  1. Delete everything on your PMS memory stick. DO NOT FORMAT!
  2. Download the following file and unzip it directly to the ROOT directory (i.e. if your memory stick is assigned drive letter G, then the root directory is G :\) of your memory stick. This is the skeleton of the Open-R SDK.

 

                     R_Code.zip

 

  1. Download the following file and unzip it to the ROOT directory of the memory stick. Overwrite ALL files if prompted.

 

                    samba.zip

 

  1. Now put back the memory stick to the AIBO and turn it on. The R-code boot sequence will begin. Once the AIBO stay idle, you will be ready for next step.
  2. Download the file below and play it in close approximation to the AIBO. You can use your PC, or convert it to MP3 and download it to a portable player of your choice.

 

                     Samba_aibo.wav

 

  1. Enjoy the show!
