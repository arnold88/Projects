Troubles Encountered

 

During the course of the project, there are several difficulties that are being encountered:

 

  1. Connecting the AIBO to the wireless

 

The AIBO in the CVRL contains a Wireless LAN card. However in order for that to connect, the access point must be set to the following:

 

 

SSID: AIBONET

WEP key: AIBO2

Wireless Channel: any channel between 1-11

LAN address: 10.0.1.1

 

 

This requires modifying the router firmware, since the wireless router in the CVRL does not support 10.x.x.x Private IP addresses. Furthermore the router connects to several production machines, and I can’t afford messing around with it.

 

  1. Loading custom software (homebrew) into PlayStation Portable (PSP)

 

During the course of the project, I need to load Lua Player to PSP so that it will interface with Webots and possibly the AIBO.

 

When purchased brand new, the PSP only allows loading of Sony-approved software. In order to load custom-made software, you must utilize the PSP service mode.

 

To enter service mode, you will need to modify your battery. Remember you need to use Genuine Sony battery, and those cost $40 a piece.

 

In the following picture I am using the PSP-2000 slim battery. If you are using some other model, use Google and look for “How to make PSP Pandora Battery”.

 

 

Here you will need to use a knife to cut the wire enclosed in the BLUE circle.

 

And then, you need to de-solder the pin that is enclosed in the RED circle.

 

Now, you have your own “Pandora Battery”. Next, you need to get a “magic memory stick”. Use Google and search for “PSP magic memory stick” for how to make one from scratch.

 

Once you have your stick, put the memory stick into PSP, put in the “Pandora Battery”, and plug in the AC adapter. Your PSP will go into service (text) mode, and after a while it should come back up with a new Operating System.

 

Congrats! You are now can run any program you want on your PSP!!
